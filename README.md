# DRAGON BALL Z

Simple hand tracking with Goku / Vegeta powers.

### How to use:
- Open `index.html` in your browser.

### Controls:
- **Left Hand Open:** Shows Goku power.
- **Right Hand Open:** Shows Vegeta power.
- **Gold Lines:** Shows the hand tracking so you know it's working.

### Note:
- Glow removed to keep it compatible with all devices.
